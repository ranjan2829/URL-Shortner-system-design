
# Quant Insider


## Cheatsheet: 35 Must-Study Models and Equations for Quantitative Finance Interviews

### Equity Models:

- [x] Lognormal Random Walk: `dS=μSdt+σSdW`
- [x] Multi-dimensional Lognormal Random Walk: `dXi=μiXidt+σiXidWi`

#### Stochastic Volatility:
- [x] Hull & White (1987)
- [x] Square-root model/Heston (1993)
- [x] 3/2 model
- [x] GARCH-diffusion
- [x] Ornstein–Uhlenbeck process
- [x] Asymptotic analysis
- [x] Schönbucher’s stochastic implied volatility

#### Jump Diffusion Model:
Incorporates jumps in asset prices.

### Fixed Income Models:

- [x] Yield to Maturity (YTM): Measure of the annual return on a bond.
- [x] Duration: Measure of the sensitivity of the price of a bond to changes in interest rates.
  - [x] Macaulay Duration
  - [x] Modified Duration
  - [x] Effective Duration
  - [x] Key Rate Duration
  - [x] Dollar Duration=Modified Duration×Change in Yield×Market Value
- [x] Spot Rate and Forward Rates: Describe the interest rate structure.
- [x] Black 1976: Pricing model for European options on bonds.
- [x] Bond Options, Caps, and Floors: Various options and derivatives on bonds.
- [x] Swaptions: Options on interest rate swaps.

### Spot Rate Models:

- [x] Vasicek
- [x] Cox, Ingersoll, and Ross
- [x] Ho and Lee
- [x] Hull and White
- [x] Black–Karasinski

### Two-Factor Models:

- [x] Brennan and Schwartz
- [x] Fong and Vasicek
- [x] Longstaff and Schwartz
- [x] Hull and White

### SABR Model:
Stochastic volatility model used to describe forward rates, not spot rates.

### Heath, Jarrow and Morton Model:
Framework for modeling the term structure of interest rates.

### Brace, Gatarek & Musiela (BGM) Model:
Extension of the Heath-Jarrow-Morton framework.

### Credit Models:

- [x] Structural Models: Represent the default or bankruptcy of a company.
  Examples include Merton's model.
